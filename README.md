# ConsumoApiLazarus

Aplicação desenvolvida em **Lazarus/Free Pascal** para demonstrar o consumo de APIs REST.  
O projeto tem como objetivo mostrar como integrar aplicações desktop com serviços externos via HTTP, utilizando bibliotecas nativas e componentes do Lazarus.

---

## 🚀 Funcionalidades
- Consumo usando o verbo HTTP (GET).
- Tratamento de respostas em **JSON**.
- Interface gráfica simples para interação com os dados.
- Estrutura modular para facilitar manutenção e expansão.

---

## 🛠️ Tecnologias utilizadas
- **Lazarus IDE**  
- **Free Pascal**  
- **FPHTTPClient** (para requisições HTTP)  
- **fpjson/jsonparser** (para manipulação de JSON)
